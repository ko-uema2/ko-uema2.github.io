# ko-uema2.github.io

Personal portfolio site built with Astro, hosted on GitHub Pages.

## Structure

```text
src/content/projects/   # Project summary markdown files
src/pages/              # Astro pages
src/layouts/            # Astro layouts
```

## Development

```bash
npm install
npm run dev
```

## Deploy

Automatically deployed to GitHub Pages via GitHub Actions on push to `main`.
